# ⚽ Soccer Free Kick 3D

ブラウザで遊べる3Dフリーキックゲームです。

## 🎮 今すぐプレイ

**[https://tomuzou.github.io/soccer-app/](https://tomuzou.github.io/soccer-app/)**

インストール不要。PC・スマホのブラウザからそのまま遊べます。

## 概要

Three.jsを使って3D空間でフリーキックを蹴るゲームアプリです。
ボールの軌道は物理エンジン（Cannon-es）でシミュレーションされます。

## 技術スタック

| 用途 | ライブラリ |
|------|-----------|
| フレームワーク | React + Vite |
| 言語 | TypeScript |
| 3Dレンダリング | Three.js |
| 物理エンジン | Cannon-es |


## ゲームモード

起動するとタイトル画面で2つのモードを選べます。

- **フリープレイ**: 好きなだけシュートしてゴール数を競うエンドレスモード
- **ステージモード**: Stage1〜10をクリアして進むミッションモード

## ゲームの遊び方

PC（マウス）・スマホ（タッチ）共通の操作です。

- **ボールを手前にドラッグ**して狙う（強く引くほどパワーUP）
- ドラッグの**左右**でコース、**引く量**で高さ（仰角）が決まる
- 画面下の **CURVE バー**を左右にスライドしてカーブ量を調整 🍌
- **指／マウスを離す**とシュート
- ゴール枠に入れて得点を狙え！

> 💡 カーブは方向操作と独立しているので、まっすぐ狙いながら大きく曲げることもできます。設定したカーブ量は次のキックにも引き継がれます。

### ステージモードについて

全10ステージ。クリアすると次のステージへ進み、進むほど難易度が上がります。
ステージごとに異なるミッションが課されます。

- ⚽ **ゴールを決める** — 基本ミッション
- 🥅 **クロスバーに当てる** — バーを狙い撃つ
- 🎯 **的・コーナーを狙う** — ゴール内の指定ゾーンを通す
- 🧱 **障害物・壁をよける** — 壁や動くキーパーをかわして決める（カーブが活躍！）

> 失敗しても何度でも挑戦できます（無制限リトライ）。条件を満たすと「STAGE CLEAR」、Stage10をクリアすると「ALL CLEAR」！

### 主な機能

- 物理エンジンによるリアルなボールの軌道
- マグヌス効果を再現したカーブシュート（方向操作とは独立したスライダーで調整）
- フリープレイ＋全10ステージのミッションモード
- パワー／カーブを表示するメーターUI
- スマホ対応（タッチ操作・レスポンシブ表示）
