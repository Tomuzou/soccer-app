## 概要

Three.jsを使って3D空間でフリーキックを蹴るゲームアプリです。
ボールの軌道は物理エンジン（Cannon-es）でシミュレーションされます。

## ゲームモード

起動するとタイトル画面で3つのモードを選べます。

- **フリープレイ**: 好きなだけシュートしてゴール数を競うエンドレスモード
- **ステージモード-α**: Stage1〜10をクリアして進む標準難度のミッションモード
- **ステージモード-β**: αより高難度。サイドネット／ポスト当て／ボールを追う**AIキーパー**などの難ミッション

## ゲームの遊び方

PC（マウス）・スマホ（タッチ）共通の操作です。

- **ボールを手前にドラッグ**して狙う（強く引くほどパワーUP）
- ドラッグの**左右**でコース、**引く量**で高さ（仰角）が決まる
- 画面下の **CURVE バー**を左右にスライドしてカーブ量を調整 🍌
- **指／マウスを離す**とシュート
- ゴール枠に入れて得点を狙え！

> 💡 カーブは方向操作と独立しているので、まっすぐ狙いながら大きく曲げることもできます。設定したカーブ量は次のキックにも引き継がれます。

### ステージモードについて

α・βともに全10ステージ。クリアすると次のステージへ進み、進むほど難易度が上がります。
ステージごとに異なるミッションが課されます。

**ステージモード-α（標準）**

- ⚽ **ゴールを決める** — 基本ミッション
- 🥅 **クロスバーに当てる** — バーを狙い撃つ
- 🎯 **的・コーナーを狙う** — ゴール内の指定ゾーンを通す
- 🧱 **障害物・壁をよける** — 壁や動くキーパーをかわして決める（カーブが活躍！）

**ステージモード-β（高難度）**

- 🥅 **サイドネットを狙う** — ゴール左右端の深い位置に突き刺す
- 🎯 **左／右ポストに当てる** — 細いポストをピンポイントで狙う
- 🧤 **AIキーパーを抜く** — ボールを追ってくるキーパー。速いシュートやカーブでないと止められる！

> 失敗しても何度でも挑戦できます（無制限リトライ）。条件を満たすと「STAGE CLEAR」、Stage10をクリアすると「ALL CLEAR」！

### 主な機能

- 物理エンジンによるリアルなボールの軌道
- マグヌス効果を再現したカーブシュート（方向操作とは独立したスライダーで調整）
- リアルなゴールネット演出 — 当たった位置を中心に波打つネット、バー／ポストの反発、得点時の発光
- フリープレイ＋2種のステージモード（α=標準 / β=高難度・AIキーパー登場）
- パワー／カーブを表示するメーターUI
- スマホ対応（タッチ操作・レスポンシブ表示）
- 成長過程アーカイブ — 初期バージョン（`/v1/`）を凍結保存し、最新版と並べて公開
