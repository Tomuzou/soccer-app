# ⚽ Soccer Free Kick 3D

ブラウザで遊べる3Dフリーキックゲームです。

## 🎮 今すぐプレイ

**[https://tomuzou.github.io/soccer-app/](https://tomuzou.github.io/soccer-app/)**

インストール不要。PC・スマホのブラウザからそのまま遊べます。

## 概要

Three.jsを使って3D空間でフリーキックを蹴るゲームアプリです。
ボールの軌道は物理エンジン（Cannon-es）でシミュレーションされます。

## 技術スタック

| 用途 | ライブラリ |
|------|-----------|
| フレームワーク | React + Vite |
| 言語 | TypeScript |
| 3Dレンダリング | Three.js |
| 物理エンジン | Cannon-es |

## セットアップ

```bash
# パッケージインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```

## ゲームの遊び方

PC（マウス）・スマホ（タッチ）共通の操作です。

- **ボールを手前にドラッグ**して狙う（強く引くほどパワーUP）
- **左右にずらす**とコースが変わる
- **引くときに弧を描く**とカーブ（バナナシュート）🍌
- **指／マウスを離す**とシュート
- ゴール枠に入れて得点を狙え！

### 主な機能

- 物理エンジンによるリアルなボールの軌道
- マグヌス効果を再現したカーブシュート
- パワー／カーブを表示するメーターUI
- スマホ対応（タッチ操作・レスポンシブ表示）
